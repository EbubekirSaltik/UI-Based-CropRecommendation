Crop Recommendation API Documentation

This API provides crop recommendations based on soil and environmental parameters.

API Endpoints

Get Crop Recommendations

GET /api/predict

Query Parameters:

  • Nitrogen - Nitrogen content in soil (required)
  • Phosporus - Phosphorus content in soil (required)
  • Potassium - Potassium content in soil (required)
  • Temperature - Temperature in Celsius (required)
  • Humidity - Humidity percentage (required)
  • Ph - pH value of soil (required)
  • Rainfall - Rainfall in mm (required)

Example Request:

GET /api/predict?Nitrogen=90&Phosporus=42&Potassium=43&Temperature=20.87&Humidity=82.00&Ph=6.50&Rainfall=202.93

Example Response:

{
  "status": "success",
  "input_parameters": {
    "Nitrogen": 90,
    "Phosporus": 42,
    "Potassium": 43,
    "Temperature": 20.87,
    "Humidity": 82.00,
    "Ph": 6.50,
    "Rainfall": 202.93
  },
  "recommendations": [
    {
      "crop": "Rice",
      "probability": 0.92
    },
    {
      "crop": "Maize",
      "probability": 0.03
    },
    ...
  ]
}

POST /api/predict

Request Body (JSON):

Error Responses

The API will return appropriate error messages with HTTP status codes:

400 Bad Request

{
  "error": "Missing parameters. Required: Nitrogen, Phosporus, Potassium, Temperature, Humidity, Ph, Rainfall"
}

or

{
  "error": "All parameters must be numeric values"
}

500 Internal Server Error

{
  "error": "Error message details"
}

Integration Example

// JavaScript fetch example
fetch('/api/predict?Nitrogen=90&Phosporus=42&Potassium=43&Temperature=20.87&Humidity=82.00&Ph=6.50&Rainfall=202.93')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
